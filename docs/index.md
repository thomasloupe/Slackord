# Slackord Documentation

Welcome to the Slackord documentation. Slackord is a .NET MAUI application that imports your Slack workspace messages into Discord.

## Features

- **Cross-Platform**: Runs on Windows and macOS
- **Full Slack Export Support**: Import messages, threads, attachments, and reactions
- **Smart Rate Limiting**: Intelligent bandwidth management to avoid Discord API limits
- **Session Management**: Resume imports where you left off
- **Slackdump Integration**: Support for Slackdump exports
